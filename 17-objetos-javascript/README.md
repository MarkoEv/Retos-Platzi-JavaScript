# Reto 17 — Objetos en JavaScript

Este archivo contiene una versión del **reto de Objetos en JavaScript** con instrucciones claras para poder practicar los ejercicios.

En algunos casos en el curso se muestra las soluciones pero no incluye el archivo del **challenge para resolverlo desde cero**, por lo que este archivo recrea el reto para poder practicar.

---

## Instrucciones

1. Copia **todo el contenido** del archivo de este repositorio.

2. Ve al archivo original del reto en tu proyecto de Platzi:

```id="e4kngb"
src/17-objetos-javascript/challenge.js
```

3. **Reemplaza completamente** el contenido de ese archivo con el código de este repositorio.

4. Guarda el archivo.

5. Ejecuta los tests:

```id="5dpdd4"
npx vitest src/17-objetos-javascript
```

Si todo está correcto, los tests deberían ejecutarse normalmente y podrás comenzar a resolver los ejercicios.

---

## Contenido del reto

Este reto practica conceptos básicos de **objetos en JavaScript**:

* Crear objetos
* Acceder a propiedades
* Acceder a propiedades dinámicas
* Verificar si una propiedad existe
* Optional chaining
* Objetos anidados
* Modificar objetos
* Crear objetos con métodos
* Uso de `this`

---

## Objetivo

Practicar cómo trabajar con **objetos y estructuras de datos en JavaScript**, que son fundamentales para el desarrollo de aplicaciones.
