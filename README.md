# Retos de JavaScript – Platzi

Este repositorio contiene archivos de retos que recrean algunos ejercicios de cursos de JavaScript en Platzi.

En algunos casos el curso solo incluye **las soluciones**, pero no el archivo del **challenge para resolverlo desde cero**.
Por eso este repositorio contiene versiones del archivo original con **instrucciones claras para practicar los ejercicios**.

---
## Detalles del curso
* Link del curso:
[Curso de Fundamentos de JavaScript](https://platzi.com/cursos/fundamentos-de-javascript/)

* Repositorio original del curso:
[Repositorio: Retos JavaScript-Platzi](https://github.com/platzi/retos-javascript-fundamentos)

* Docente del curso:
**Oscar Barajas Tavares**  
  [gndx](https://github.com/gndx)

## Objetivo

Permitir que los estudiantes de Platzi puedan:

* Resolver los ejercicios desde cero
* Practicar lógica en JavaScript
* Entender mejor los conceptos del curso

---

## Cómo usar los retos

1. Entra al archivo del reto dentro del repositorio.

2. Copia **todo el contenido** del archivo.

3. Ve al archivo original del curso en tu proyecto.

Ejemplo:

```
src/15-closures-javascript/challenge.js
```

4. **Reemplaza completamente el contenido** con el archivo de este repositorio.

5. Guarda el archivo.

6. Ejecuta los tests del proyecto:

```
npx vitest src/15-closures-javascript
```
