# Reto 16 — Arrays en JavaScript

Este archivo contiene una versión del **reto de Arrays** con instrucciones claras para poder practicar los ejercicios.

En algunos casos el curso muestra las soluciones pero no incluye el archivo del **challenge para resolverlo desde cero**, por lo que este archivo recrea el reto para poder practicar.

---

## Instrucciones

1. Copia **todo el contenido** del archivo de este repositorio.

2. Ve al archivo original del reto en tu proyecto de Platzi:

```
src/16-arrays-javascript/challenge.js
```

3. **Reemplaza completamente** el contenido de ese archivo con el código de este repositorio.

4. Guarda el archivo.

5. Ejecuta los tests:

```
npx vitest src/16-arrays-javascript
```

Si todo está correcto, los tests deberían ejecutarse normalmente y podrás comenzar a resolver los ejercicios.

---

## Contenido del reto

Este reto practica operaciones comunes con **arrays en JavaScript**:

* Crear arrays
* Acceder a elementos
* Agregar elementos
* Reemplazar valores
* Eliminar elementos
* Obtener longitud
* Verificar si un valor es un array
* Encontrar índices

---

## Desafío final

El último ejercicio practica un **CRUD completo sobre arrays**, aplicando operaciones como:

* `push`
* `unshift`
* `splice`
* `pop`
