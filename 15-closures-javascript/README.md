# Reto de Closures en JavaScript

Este archivo contiene una versión del **reto de closures** con instrucciones más claras para poder practicar los ejercicios.

El profesor publicó las soluciones pero no incluyó el archivo del reto, por lo que este archivo recrea los ejercicios para poder resolverlos desde cero.

## Instrucciones

1. Copia **todo el contenido** del archivo de este repositorio.

2. Ve al archivo original del reto en tu proyecto de Platzi:

```
src/15-closures-javascript/challenge.js
```

3. **Reemplaza completamente** el contenido de ese archivo con el código de este repositorio.

4. Guarda el archivo.

5. Ejecuta los tests:

```
npx vitest src/15-closures-javascript
```

Si todo está correcto, los tests deberían ejecutarse normalmente y podrás comenzar a resolver los ejercicios.

## Objetivo

Practicar:

* Closures
* Estado privado
* Encapsulación en JavaScript
* Manejo de objetos y arrays

Todos los retos están organizados por niveles de dificultad para facilitar el aprendizaje.
